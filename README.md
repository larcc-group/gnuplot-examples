# Gnuplot Examples

This repository has some tips on how to use Gnuplot.


# Dependencies

Install the following dependencies to use Gnuplot.

```console
foo@bar:~$ sudo apt install gnuplot
```


# Generating plot

```console
foo@bar:~$ gnuplot script-name.gp
```
